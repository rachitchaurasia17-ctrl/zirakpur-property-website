# Sachdeva Real Estate

Marketing site for **Sachdeva Real Estate** — property consultants & investment experts to the Mohali tri-city.

Static React app (React 18 + Babel standalone, no build step). Open `index.html` to run, or serve the folder with any static server.

- `index.html` — shell, fonts, global styles, script tags
- `app.jsx` — all UI sections (nav, hero, portfolio, services, about, contact, footer)
- `properties.js` — listing catalogue (`window.RS_PROPERTIES`)
- `components/IntroTunnel.jsx` + `.css` — cinematic 3D tunnel intro
- `assets/` — imagery
